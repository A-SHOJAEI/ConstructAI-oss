# Demo Photos

Sample construction photos for the ConstructAI demo environment.

## Generation

Run `bash download_sample_photos.sh` to generate placeholder images.
Requires Pillow (`pip install Pillow`).

## For Production Demos

Replace placeholder images with actual construction site photos.
Ensure you have appropriate rights/licenses for any photos used.

## Suggested Sources (Public Domain / CC0)

- [Unsplash Construction Collection](https://unsplash.com/s/photos/construction)
- [Pexels Construction](https://www.pexels.com/search/construction/)
- [Pixabay Construction](https://pixabay.com/images/search/construction/)

All images in this directory should be 1920x1080 JPEG for consistency
with the camera feed simulation.
