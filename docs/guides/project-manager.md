# Project Manager Guide

## Overview
ConstructAI provides AI-powered tools to help project managers track progress,
manage costs, and make data-driven decisions.

## Key Features

### Dashboard
View real-time project health with SPI/CPI indicators, safety alerts,
and quality metrics at a glance.

### Document Search
Ask questions about project documents using natural language:
- "What are the concrete specifications for Building A?"
- "Show me the electrical submittal requirements"
- "What does the contract say about liquidated damages?"

### Cost Tracking
Monitor cost performance with EVM metrics:
- **SPI** (Schedule Performance Index): On schedule if >= 1.0
- **CPI** (Cost Performance Index): On budget if >= 1.0
- **EAC** (Estimate at Completion): Projected final cost

### Daily Reports
Auto-generated daily reports aggregating:
- Weather conditions and work hours
- Safety incidents and observations
- Quality inspections completed
- Productivity metrics by trade

### Change Orders
Submit and track change orders with AI-assisted impact analysis:
- Automatic cost impact estimation
- Schedule impact assessment
- Risk exposure calculation
