# Superintendent Guide

## Overview
ConstructAI helps superintendents manage daily operations, track productivity,
and coordinate subcontractors.

## Key Features

### Daily Log
Record daily activities with AI assistance:
- Weather conditions auto-populated from local weather API
- Crew counts and work areas
- Equipment usage and idle time
- Material deliveries and issues

### Productivity Tracking
Monitor crew productivity in real-time:
- Activity recognition from site cameras
- Comparison to baseline productivity rates
- Trend analysis with forecasting

### Schedule Management
View and update the project schedule:
- Critical path activities highlighted
- Float consumption tracking
- Three-week lookahead generation
- Weather delay impact analysis

### Quality Inspections
Manage quality inspections:
- AI-assisted defect detection from photos
- Punch list management
- NCR (Non-Conformance Report) tracking
- Compliance checklist automation
